## ✨ Features

- **CRT Flicker Effect** — Authentic old-school CRT screen simulation
- **Retro Fonts** — Classic typefaces that evoke the golden era of computing
- **Notification Sounds** — Vintage audio cues for an immersive experience
- **Fully Responsive** — Works seamlessly across all screen sizes

## 🚀 Installation

1. Open your Docker container's terminal and run:

   ```bash
   git clone https://github.com/jagdishtripathy/pixo.git /opt/CTFd/CTFd/themes/pixo
   ```

2. Log in as **Admin** and navigate to:

   > **Admin Panel → Config → Themes**

3. Select **pixo** from the theme dropdown and click **Update**.

You're all set! 🎉
